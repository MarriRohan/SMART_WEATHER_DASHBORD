# SMART_WEATHER_DASHBORD
## Objective
A web app that fetches and displays live weather details (temperature, humidity, wind speed, forecast) using the OpenWeather API.

## Features
- Supports multiple cities
- Shows weather icons and forecast trends
- Responsive design with HTML/CSS
- Hosted on GitHub Pages

## API Integration
- Current weather: `https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}`
- Forecast: `https://api.openweathermap.org/data/2.5/forecast?q={city}&appid={API_KEY}`
- Data returned in JSON format, parsed using JavaScript.

## Deployment
1. Push code to GitHub.
2. Enable GitHub Pages in repository settings.
3. Access app at `https://username.github.io/smart-weather-dashboard`.
